Die App.
Diese App installiert eine angepasste gps.conf (GPS Konfigurations-) Datei für ihr GPS. Sie bietet mehrere Profile zur Auswahl. Wir empfehlen, das standard Google Profil zu nutzen , aber es steht ihnen natürlich frei, ein anderes Profil zu wählen. Abgesehen von den allgemeinen Profilen können sie das von ihnen gewählte Profil an ihre Bedürfnisse anpassen.

NTP Server
Derek Gordon unterhält einen spezialisierten NTP Server Pool für sie. Entgegen der üblichen Meinung ist es nicht immer der örtlich nächst gelegene Zeitserver, der am besten ist. Wir sehen Derek Gordons Server als die beste Option weil es die besten Daten hat und am häufigsten aktualisiert wird.
Für weiterführende Inforamtionen zu den NTP Servern lesen sie die "Über diese App" Sektion, welche sie in den Einstellungen dieser App finden. Sie können sich auch gerne auf derekgordon.com nach weiterführenden Informationen umschauen.

aGPS Almanach
Derek Gordon unterhält einen speziellen aGPS Almanach für Geräte die Qualcomm oder vergleichbare Chipsets benutzen. Diese xtra.bin genannte Datendatei wird alle 30 Minuten aktualisiert und befindet sich auf einem Verteilersystem, dass von der großzügigen Nutzergemeinde von Derek Gordon finanziert wird.
Für weiterführende Inforamtionen zu den NTP Servern lesen sie die "Über diese App" Sektion, welche sie in den Einstellungen dieser App finden. Sie können sich auch gerne auf derekgordon.com nach weiterführenden Informationen umschauen.

Sichern und Wiederherstellen
Während des ersten Starts der App wird eine Sicherung auf der SD-Karte angelegt. Löschen sie es nicht weil eine Sicherung nur während des ersten Starts der App angelegt wird (oder bei einem Update, falls sie es doch entfernt haben) und es sonst keine weitere Sicherung auf der SD-Karte gibt.

ROM Aktualisierungen
Der Patch den diese App anbietet befindet sich in ihrer /system Partition, also wird der Patch entfernt sobald sie ihren ROM aktualisieren. Sie müssen den Patch dann erneut installieren.
Im Fall dass sie einen Cyanogenmod basierten ROM mit addon.d Unterstützung nutzen wird ein Script im addon.d Verzeichnis erstellt, was den Patch erhält, dann müssen sie bei einer ROM Aktualisierung den Patch nicht neu installieren.
Es kann in den App Einstellungen deaktiviert werden, ist aber standardmäßig aktiviert.

Asus Nutzer
Bei ASUS ROMs gibt es eine typische Datei im Verzeichnis /system/etc/gps, namentlich die gpsconfig.xml Datei. Diese App verändert diese Datei ebenfalls, um genau zu sein den PeriodicTimeOutSec Parameter. Selbstverständlich wird auch von dieser Datei eine Sicherung angelegt.
